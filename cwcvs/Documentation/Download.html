CWCVS Download

Current Version.


The current Carbon version of CWCVS is 2.4. The last available Classic version is 2.3.2. Check this page for future releases.

New features in version 2.4 include:

  • Carbonize;
  • Detect newer versions of unmergable files;

The download file is a StuffIt archive containing CWCVS, VCS Environment, a copy of the User's Manual, the current Release Notes, and a Bug Report form. CWCVS also comes with CWProjector, an MPW Projector plugin. There are also a Press Release and a FAQ for the product.

  • Download CWCVS 2.4 and CWProjector 3.4 (BinHex)
  • Download CWCVS 2.4 and CWProjector 3.4 (MacBinary).
  • Download CWCVS 2.3.2 (BinHex)

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online through Kagi. Upgrades are free for registered users, and CWProjector uses the same registration codes, so users of one product may use the other.
