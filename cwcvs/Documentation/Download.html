CWCVS Download

Current Version.


The current version of CWCVS is 1.0. Check this page for future releases.

Version 1.0 features include:

  • Basic support for update, commit, log, status, diff, add and remove for individual files;
  • Basic support for update, commit and tag for the entire source tree;
  • Implementation of the Rename operation;
  • Locking of files by default;
  • KClient support for use with Kerberos;
  • Customizable environment variable settings;

The download file is a binhexed StuffIt! archive containing CWCVS, VCS Environment, a copy of the User's Manual, the current Release Notes, and a Bug Report form. CWCVS also comes with CWProjector, an MPW Projector plugin. There are also a Press Release and a FAQ for the product.

Download CWCVS 1.0 and CWProjector 3.0.

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online through Kagi. Upgrades are free for registered users, and CWProjector uses the same registration codes, so users of one product may use the other.

Prerelease Version.

The current prerelease version of CWCVS is 2.0b1. New features include:

  • Conversion to latest CodeWarrior API;
  • Advanced dialogs for most commands;
  • Recursive command versions for Difference, History and Status;
  • rtag support in the Label command;
  • Default to -kb for adding non-text files
  • Automatic login upon connect;
  • Automatic directory creation on Add;
  • Watchers support;
  • New default options preferences panel;
  • Various bug fixes;

The download file is a binhexed StuffIt! archive containing CWCVS 2.0b1, the VCSEnvironment Pref preferences panel, a copy of the User's Manual, the current Release Notes, and a Bug Report form.

Download CWCVS 2.0b1.

The alpha designation mostly means that the feature set is incomplete and the beta designation means that we consider the product feature complete but there may be a few small glitches left that we missed. CWCVS is used for its own development, so hopefully bugs should be minimal at all times. We always welcome feedback and suggestions.
