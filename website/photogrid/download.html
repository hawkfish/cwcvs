Reporting Bugs

Please run SoloBug for PhotoPress. For each bug, create a new report, enter as many details as possible, save the report, and e-mail it to us as an attachment at photogrid-bugs@electricfish.com. (Please send each report as a separate bug file.)

System Requirements

Power Macintosh (or compatible)
32 megabytes of RAM
MacOS 8.6 or later
CarbonLib 1.2 or later
QuickTime 4.0 or later
