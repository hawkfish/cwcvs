PhotoGrid

Download from our server (1.9 MB, compressed with StuffIt). The current version is 1.0a17 (released 7 August).

We continue to polish PhotoGrid, releasing a new version every few days. Please check back here periodically to see if a newer version is available.

Reporting Bugs

Please run SoloBug for PhotoGrid. For each bug, create a new report, enter as many details as possible, save the report, and e-mail it to us as an attachment at photogrid-bugs@electricfish.com. (Please send each report as a separate bug file.)

System Requirements

Power Macintosh (or compatible)
32 megabytes of RAM
MacOS 8.6 or later
CarbonLib 1.2 or later
QuickTime 4.0 or later
