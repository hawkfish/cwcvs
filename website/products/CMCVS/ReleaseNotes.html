1.0a5

I have added Finder labels to CWCVS and this has percolated into CMCVS. In order to support the labels from the menu, I have added the Synchronize Status command to the top of the menu. Note that for single files this will only update the status of that one file; likewise, to update the labels for a directory, use the command for the entire directory tree.

Change history:

  • Use Finder labels;
  • Add Synchronize Status commands;

Return to the Index.

1.0a4

I got tired of having to configure a tool that has its default configuration information readily available, so for 1.0a4 I have added an auto-configure facility. The CVS Options are set to the defaults and the VCS Environment is initialized with the CVSROOT value stored in the CVS:Root file. If you wish to change these options, you can still use CodeWarrior, but non-CW projects can now be worked with more readily.

Change history:

  • Autoconfigure if no project file present;
