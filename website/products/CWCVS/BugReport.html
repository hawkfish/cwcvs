CWCVS Bug Report Form

If you are using a prerelease version of CWCVS, there should be a file called CWCVS.log in the preferences folder containing a time stamped summary of the plugin/client traffic. Please extract the relevant lines (if any) and include them in the report.

Also, if this is actually a bug in MacCVS, please use the WinCVS.org Bugzilla system.
