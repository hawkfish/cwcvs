CWCVS

Description.

CWCVS is a CodeWarrior version control plug-in which provides access to the GNU CVS system from inside the CodeWarrior IDE. CVS is a widely used POSIX-based version control system and CVS clients are available on many platforms, including the Macintosh, making it a reasonable choice for multiple platform development. CWCVS allows Macintosh developers to use CVS with CodeWarrior without arcane command line arguments or scripts.

CWCVS is shareware and you can download it right now and try it out.

Features.

CWCVS includes the following features:

  • Access to most commands with defaults or customization dialogs;
  • Recursive versions of most commands;
  • Default to -kb for adding non-text files
  • Automatic login upon connect;
  • Automatic directory creation on Add;
  • Watchers support;
  • Implementation of the Rename operation;
  • Locking of files by default;
  • KClient support for use with Kerberos;
  • Customizable environment variable settings;
  • Customizable default options preferences panel;

CWCVS also comes with CWProjector, a MPW Projector plugin.

Requirements.

CWCVS requires CodeWarrior IDE 1.7.4 or later. You will also need WinCvs.org's MacCVS application.
