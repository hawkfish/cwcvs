Current Version.


The current Carbon version of CWProjector is 3.4. The last available Classic version is 3.3. Check this page for future releases.

New features for version 3.4 include:

  • Carbon conversion;
  • OSX;
  • Various bug fixes;

The download file is a StuffIt! archive containing CWProjector, the VCSComment Pref and CWProjector Options preferences panels, a copy of the User's Manual, the current ReleaseNotes , and a Bug Report form. CWProjectoralso comes with CWCVS, a CVS plugin. There are alsoa Press Release and a FAQfor the product.

For IDE 4.1 and later:

  • Download CWProjector 3.4 and CWCVS 2.3.2 (BinHex)
  • Download CWProjector 3.4 and CWCVS   2.3.2 (MacBinary)
For ID 4.0.4 and earlier:
  • Download CWProjector 3.3 (BinHex)

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online throughKagi. Upgrades are free for registered users, and CWCVS uses the same registration codes, so users of one product may use the other.
