CWProjector Download

Current Version.


The current version of CWProjector is 3.3. Check this page for future releases.

New features for version 3.3 include:

  • Optional clipboard checkin comments;
  • Retry SourceServer when busy;
  • Various bug fixes;

The download file is a StuffIt! archive containing CWProjector, the VCSComment Pref and CWProjector Options preferences panels, a copy of the User's Manual, the current Release Notes, and a Bug Report form. CWProjector also comes with CWCVS, a CVS plugin. There are also a Press Release and a FAQ for the product.

  • Download CWProjector 3.3 and CWCVS 2.3 (BinHex)
  • Download CWProjector 3.3 and CWCVS 2.3 (MacBinary).

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online through Kagi. Upgrades are free for registered users, and CWCVS uses the same registration codes, so users of one product may use the other.

Prerelease Version.

The current prerelease version of CWProjector is 3.4b1. New features in 3.4 include:

  • Carbon build;
  • OSX;
  • Various bug fixes;

The download file is a binhexed StuffIt! archive containing CWProjector 3.4b1, the VCSComment Pref and CWProjector Options preferences panels, a copy of the User's Manual, the current Release Notes, and a Bug Report form.

Download CWProjector 3.4b1.

The alpha designation mostly means that the feature set is incomplete and the beta designation means that we consider the product feature complete but there may be a few small glitches left that we missed. CWProjector is used for its own development, so hopefully bugs should be minimal at all times. We always welcome feedback and suggestions.
