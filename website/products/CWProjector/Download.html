CWProjector Download

Current Version.


The current version of CWProjector is 3.0. Check this page for future releases.

New features for version 3.0 include:

  • Difference between local file and any version;
  • Get enabled for orphaned/missing files;
  • Export/import support via the Purge (OrphanFiles) and Add commands;
  • Recursive versions of most commands;
  • Improved Task Name support in Add, Checkout, Checkin and Comment;
  • Full support for reporting (ProjectInfo) options;
  • Improved Label support;
  • Prefs panels for specifying default dialog options ;
  • Backward compatible with all VCS enabled IDEs.
  • Checkin revision specification;
  • Various bug fixes;

The download file is a binhexed StuffIt! archive containing CWProjector, VCSComment Pref, a copy of the User's Manual, the current Release Notes, and a Bug Report form. CWProjector also comes with CWCVS, a CVS plugin. There are also a Press Release and a FAQ for the product.

Download CWProjector 3.0 and CWCVS 1.0.

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online through Kagi. Upgrades are free for registered users, and CWCVS uses the same registration codes, so users of one product may use the other.

There is no current prerelease version of CWProjector. Watch this space for future prereleases.
