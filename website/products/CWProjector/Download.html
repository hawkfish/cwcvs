CWProjector Download

Current Version.


The current version of CWProjector is 3.1.1. Check this page for future releases.

New features for version 3.1.1 include:

  • Helper applications for Difference;
  • Various bug fixes;

The download file is a binhexed StuffIt! archive containing CWProjector, the VCSComment Pref and CWProjector Options preferences panels, a copy of the User's Manual, the current Release Notes, and a Bug Report form. CWProjector also comes with CWCVS, a CVS plugin. There are also a Press Release and a FAQ for the product.

Download CWProjector 3.1.1 and CWCVS 2.0.

CWProjector and CWCVS are distributed as US$20 shareware and will prompt you for a serial number when you load the plugin. You can register your version using the enclosed Register application, or you can register online through Kagi. Upgrades are free for registered users, and CWCVS uses the same registration codes, so users of one product may use the other.

There is no current prerelease version of CWProjector. watch this space for future releases.
