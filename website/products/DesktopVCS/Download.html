Prerelease Version.

The current prerelease version of Desktop VCS is 1.0a5, and we are initially requiring you to be a registered CWProjector or CWCVS user to test the product. This will change for the beta release.

New features in 1.0a5 include:

  • Add helper difference to CMCVS;
  • Fix search failure bug;

The download file is a binhexed StuffIt! archive containing the Desktop VCS 1.0a5 installer.

Download the Desktop VCS 1.0a5 installer.

The alpha designation mostly means that the feature set is incomplete and the beta designation means that we consider the product feature complete but there may be a few small glitches left that we missed. Desktop VCS is used for its own development, so hopefully bugs should be minimal at all times. We always welcome feedback and suggestions.
