1.0a5

There wasa problem in both plugins that if a configuration file contained bad data (i.e. invalid aliases) the search would be aborted. This has been fixed.

Change history:

  • Fix search abort problem in both menus;
  • Convert to latest PowerPlant;
  • Convert to CarbonLib 1.0.2;


Return to the Index.

1.0a4

I have expanded the range of Finder file selections that CMProjector and CMCVS now work with. In earlier versions they would only work with single files; now they work with either a single folder or a list of files from the same folder. This should make using them a lot more efficient.

Change history:

  • Support for file list contexts in CMProjector and CMCVS;
  • Removed autoconfigure code from CMCVS;
  • Add Drag and Drop support to the path controls;


Return to the Index.
