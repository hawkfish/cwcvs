NewsWatcher-X

Description

NewsWatcher-X is a carbonized version of John Norstad's NewsWatcher program. NewsWatcher-X runs under Mac OS X or with CarbonLib 1.0.4. or later. Have a look at the screen shot to see what it looks like under OS X.

NewsWatcher-X is available as freeware and you can download it from our web site. The source code is also available, and requires CodeWarrior Pro 7.

Functionality

NewsWatcher-X works, but is not the most feature rich application you will ever see. I developed it under OS 9.1 and have tested it under OS X and OS X.1. Not all the functionality has been tested by me, but judging by the number of downloads (over 20,000) and bug reports (under 30) I believe it works quite well for what it does. I try to respond promptly to serious problems.

Documentation

The NewsWatcher 2.2.2 manual is available in both stand alone postcard format and the proprietary Microsoft Word format.

Problems

Check out the problems page,which has a list of known problems, a list of fixed problems and directions for filing bug reports.
