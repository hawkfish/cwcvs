NewsWatcher-X Problems

Reporting Problems

All reported problems are being tracked with a BugLink database. Before reporting problems, please check the known problems and check what my definition of a problem is. Also, if you notice some resource related layout glitches, fixed resources would be greatly appreciated.

The first choice is to use BugLink Reporter. Just download the application and run it. You can then use BugLink Reporter to file bug reports which I can import easily into BugLink.

You can also send reports directly to me if BugLink Reporter is too much of a pain to use.

What constitutes a problem?

I am trying not to add features to NewsWatcher-X. That has already been done admirably by Simon Fraser, Brian Clark and others who are Carbonizing as we speak. So unless it severely impacts the functionality (e.g. Printing and Navigation Services) the rule is that if the old NewsWatcher (2.2.2) works that way, so does NewsWatcher-X. I am happy to listen to requests, but this is the basic guideline.

Possible problems are ones that I cannot reproduce. Any details would be helpful.
