NewsWatcher-X Problems

Reporting Problems

All reported problems are being tracked with a BugLink database. Before reporting problems, please check the known problems. Also, if you notice some resource related layout glitches, fixed resources would be greatly appreciated.

The first choice is to use the BugLink Reporter document which requires the PandaWave's BugLink Reporter application. Copy the NewsWatcher-X BugLink Reporter document into the Data folder next to the BugLink Reporter application. You can then use BugLink Reporter to file bug reports which I can import easily into BugLink.

You can also send reports directly to me if BugLink Reporter is too much of a pain to use.
