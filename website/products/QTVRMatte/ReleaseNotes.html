QTVR Matte Release Notes

2.0 (pre-release)

  • rewritten for AE5 sdk.
  • fixed a bug which caused a crash when hitting "cancel" from file dialog
  • renders cubics correctly (but you must choose "Normal-Quality" codec
  • renders object-movies!!

 

1.0.2

  • Fixed a bug which was preventing the viewing of large Field of View values.

 

1.0.1

  • First Release Version!!
  • Initial Field-of-View more reasonable (20 degrees)
  • Serial number required to enable full functionality.
    Note: serial numbers will soon be automatically generated when you purchase the product
    from Kagi, but you may have to obtain a serial number directly from me (dav) for a few days.

    If you have any problems obtaining a serial-number, or using a serial number which was generated by Kagi, contact me.
