Description

Color Pal is an Adobe After Effects plug-in which provides a persistent customizable floating color palette within After Effects. Color Pal allows you to easily store and access a large set of standard colors for use in your projects.

Return to Contents

Features

  • A set of eight editable color chips that are readily available.
  • A list of 256 editable color chips that can be scrolled and resized.
  • Colors are saved between sessions.

Requirements

  • MacOS or Microsoft Windows based computer.
  • Adobe After Effects 5.0 or higher.

Installation

Download the version appropriate for your operating system and unpack it. Place the "Color Pal" folder (which contains the Color Pal plug-in) into the "Plug-ins" folder in your After Effects folder. You must restart After Effects for the plug-in to be available. Color Pal should be visible in the "Window" menu.

Use

To activate Color Pal, simply choose it from the Window menu. There are two parts to the Color Pal palette: the main palette and the "hot chips".

The main palette is a scrolling list of 256 color chips at the top of the Color Pal window. These come initialized to a selection of colors in the RGB color space. To change one of the chips, simply click on it to bring up the color chooser, select a new color and click OK.

The "hot chips" are a row of eight colored rectangles along the bottom of the Color Pal window. These come initialized to the primaries white, black, red, green, blue, cyan, magenta and yellow. To change one of them, simply click on it to bring up the color chooser, select a new color and click OK.

The window can be resized to reduce the number of rows. When the number of rows displayed in the main palette is smaller than the number in the list, a scroll bar will appear to allow access to all the colors, unless you have reduced the palette to the minimum size, in which case only the hot chips will be visible.

When you quit After Effects, the colors you were using will be saved automatically so that they will be available when you start your next session.
