Products.


Electric Fish, Inc. is proud to offer the following products:

  • CWProjector. A version control plugin for using MPW Projector with Metrowerks CodeWarrior.
  • CWCVS, a version control plugin plugin for using GNU CVS with Metrowerks CodeWarrior.
  • QTVR Matte. QTVR Matte is a plug-in for Adobe® After Effects which allows you to manipulate Quicktime VR panoramas from within the After Effects Environment.
  • NewsWatcher-X. A carbonized version of John Norstad's NewsWatcher.
  • Sherlock Plugins. Free Sherlock plugins for Macintosh development and other tasks.
