Electric Fish Sherlock Plugins

I have taken to writing Sherlock plugins whenever I find myself searching an online database frequently. As I use these all the time they will probably be kept up to date. Feel free to use them.

Macintosh Programming Newsgroups.

This plugin searches the Deja (formerly DejaNews) usenet database. The search is restricted to the Macintosh development newsgroups.

  • Download MacDevNews.src

Microsoft Developer Network.

This plugin searches the Microsoft Developer Network online database. This can be very useful when looking at a pile of Windoze gobbldygook that you are attempting to port.

  • Download MSDN.src

Webcrawler.

This plugin searches the Webcrawler online database. Despite having a nice results page with relevance rankings, they didn't have a Sherlock plugin. Now they do...

  • Download WebCrawler.src
