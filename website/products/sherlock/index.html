Electric Fish Sherlock Plugins

I have taken to writing Sherlock plugins whenever I find myself searching an online database frequently. As I use these all the time they will probably be kept up to date. Feel free to use them.

Search Engines.

These plugins are for search engines that don't have Sherlock plugins. Sherlock is really great for doing metasearching (searching many indexes at once) and adding these ones should increase your search power significantly.

  • Google.src searches web using the Google engine. Relevance ranking implemented.
  • WebCrawler.src searches web using the Webcrawler engine. Relevance ranking implemented

Newsgroups.

These plugins search the Deja (formerly DejaNews) usenet database. They are all software development related.

  • MacDevNews.src searches the comp.sys.mac.programmer.* newsgroup hierarchy. These groups discuss Macintosh programming.
  • CPlusPlusNews.src searches the comp.lang.c++ moderated newsgroup. This group discusses C++ programming.
  • VCSNews.src searches the comp.sys.config-mgmt newsgroup. This group discusses systems for configuration management (version control, bug tracking, etc.).

Programming.

These plugins are for searching other online programming resources.

  • MSDN.src searches the Microsoft Developer Network online database. This can be very useful when looking at a pile of Windoze gobbldygook that you are attempting to port.

Miscellaneous.

These plugins are ones that I made for fun, just to see what one could do with Sherlock.

  • eBay.src searches the eBay online auction.
